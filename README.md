## Features & Goals

ADAPT (Automatic Development of Assessment and Psychology Tests) is a tool that help researchers generate
test items, revise item set and evaluate test with psychometric theories.

By just inputting a prompt describing the test user wants to create, AI agents research around the construct,
create a pool of test items, revise them base on theoretical, linguistic (readability), and inclusion (bias) considerations
, and output a complete set of questions, ready for psychometric evaluation.
